# 17 - Hipotenusa do triangulo

## 🎯 Objetivo

Calcular a hipotenusa a partir dos catetos informados.

## 🧠 O que pratiquei

- Funcao `hypot()` do modulo `math`
- Teorema de Pitagoras
- Leitura de medidas em ponto flutuante
