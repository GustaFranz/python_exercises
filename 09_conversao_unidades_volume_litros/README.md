# 09 - Conversao de volume em litros

## 🎯 Objetivo

Converter litros para outras unidades de volume.

## 🧠 O que pratiquei

- Conversao entre kl, hl, dal, l, dl, cl e ml
- Entrada de numero decimal
- Exibicao de varios resultados com `print()`

## 🧾 Código

Arquivo: [`main.py`](https://github.com/GustaFranz/exercicios_python/blob/main/09_conversao_unidades_volume_litros/main.py)

## 🚀 Como executar


```bash
cd "09_conversao_unidades_volume_litros"
python main.py
```
