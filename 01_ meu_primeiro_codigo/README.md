# 01 - Meu primeiro codigo

## 🎯 Objetivo

Coletar informacoes basicas do usuario e exibir tudo em uma unica frase.

## 🧠 O que pratiquei

- Uso de `input()` para ler textos e numeros
- Conversao de tipos com `int()`
- Concatenacao de strings com `print()`

```bash
cd "01_ meu_primeiro_codigo"
python main.py
```
