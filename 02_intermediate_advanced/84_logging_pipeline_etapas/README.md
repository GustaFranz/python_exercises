# 84 - Logging: pipeline por etapas

## Objetivo

Registrar cada etapa de um pipeline de processamento.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | LimpezaDados Servicos |
| **Setor** | Tratamento de dados |
| **Solicitacao** | Rastrear etapas de limpeza de base de clientes importada. |

## Enunciado

- Implemente pipeline de 3 etapas com log em cada uma.
- Processe lista com entradas vazias e exiba quantidade final.

## Como executar

```bash
cd "84_logging_pipeline_etapas"
python main.py
```
