# 02 — Intermediate Advanced

Trilha de exercicios em nivel **intermediario**: mais desafiadores que os fundamentos, mas sem complexidade excessiva. Consolida logica, organizacao de codigo e problemas um pouco mais proximos de situacoes reais.

**Total:** 125 exercicios publicados (`01` a `125`) — trilha completa.

[← Voltar ao README principal](../README.md) · [Fundamentos Python](../01_python_fundamentals/) · [Bibliotecas e automacao](../03_python_libraries/) · [Materiais PDF](../04_study_materials/)

---

## Objetivo desta area

| Item | Descricao |
|------|-----------|
| **Para quem** | Quem ja concluiu (ou domina) a trilha `01_python_fundamentals` |
| **Nivel** | Intermediario — desafio moderado, sem projetos muito grandes |
| **O que treina** | Combinacao de conceitos, refatoracao, validacao e pequenos sistemas |
| **Cada exercicio** | Pasta `NN_snake_case/` com `README.md` e `main.py` |

---

## Pre-requisitos recomendados

Antes de comecar esta trilha, vale estar confortavel com:

- funcoes (incluindo `*args`, `**kwargs` e lambdas)
- listas, tuplas e dicionarios
- modulos e tratamento basico de excecoes
- leitura e escrita simples de arquivos

---

## Como executar

```bash
git clone https://github.com/GustaFranz/python_exercises.git
cd python_exercises/02_intermediate_advanced
cd 01_nome_do_exercicio    # quando os exercicios forem publicados
python main.py
```

1. Entre na pasta do exercicio
2. Leia o `README.md` da pasta
3. Execute `python main.py`
