# 118 - Assert: funcao de desconto

## Objetivo

Testar funcao de desconto com asserts em multiplos cenarios.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Loja Virtual Escolar |
| **Setor** | Varejo / precificacao |
| **Solicitacao** | Validar regras de desconto progressivo antes da campanha do bazar. |

## Enunciado

- Implemente aplicar_desconto(valor, pct) com validacao de pct.
- Teste desconto valido, zero e pct invalido com assert.

## Como executar

```bash
cd "118_assert_funcao_desconto"
python main.py
```
