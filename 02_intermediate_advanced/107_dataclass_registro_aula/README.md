# 107 - Dataclass: RegistroAula

## Objetivo

Modelar registro de aula com dataclass e campos de data.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Edutech Brasil |
| **Setor** | Educacao / diario de classe |
| **Solicitacao** | Registrar aulas ministradas com estrutura padronizada. |

## Enunciado

- Crie dataclass RegistroAula com 4 campos.
- Cadastre 2 aulas e exiba os registros.

## Como executar

```bash
cd "107_dataclass_registro_aula"
python main.py
```
