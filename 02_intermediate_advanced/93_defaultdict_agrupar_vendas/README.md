# 93 - Defaultdict: agrupar vendas

## Objetivo

Agrupar vendas por vendedor com defaultdict.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | FinEdu Carteira |
| **Setor** | Financeiro educacional |
| **Solicitacao** | Consolidar vendas de material didatico por representante. |

## Enunciado

- Agrupe vendas por vendedor com defaultdict(float).
- Exiba total de cada vendedor.

## Como executar

```bash
cd "93_defaultdict_agrupar_vendas"
python main.py
```
