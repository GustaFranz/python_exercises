# 88 - HTTP: mensagem amigavel

## Objetivo

Traduzir erros tecnicos HTTP em mensagens para o usuario final.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Secretaria Digital |
| **Setor** | Educacao / atendimento |
| **Solicitacao** | Exibir mensagens claras quando o portal de notas estiver fora do ar. |

## Enunciado

- Implemente mensagem_para_usuario(resposta) com 3 mensagens amigaveis.
- Teste status 404, 500 e timeout (0).

## Como executar

```bash
cd "88_http_mensagem_amigavel"
python main.py
```
