# 89 - HTTP: consulta com fallback

## Objetivo

Consultar API simulada de clima com dados padrao em caso de falha.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | AgroEscola |
| **Setor** | Educacao / campo |
| **Solicitacao** | Exibir previsao do tempo para visita tecnica mesmo se API falhar. |

## Enunciado

- Implemente obter_previsao com API simulada e fallback.
- Exiba fonte dos dados (api ou cache local).

## Como executar

```bash
cd "89_http_clima_fallback"
python main.py
```
