# 70 - Closure: fabrica de validadores

## Objetivo

Criar fabrica que gera validadores com min e max configuraveis.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Clinica BemViver |
| **Setor** | Saude / triagem |
| **Solicitacao** | Validar sinais vitais com faixas diferentes por tipo de exame. |

## Enunciado

- Implemente criar_validador(minimo, maximo).
- Teste validadores de pressao (90-140) e temperatura (35.0-37.5).

## Como executar

```bash
cd "70_closure_fabrica_validadores"
python main.py
```
