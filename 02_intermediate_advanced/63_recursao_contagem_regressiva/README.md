# 63 - Recursao: contagem regressiva

## Objetivo

Imprimir contagem regressiva de N ate 0 com recursao.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | LogiRapida |
| **Setor** | Logistica / expedicao |
| **Solicitacao** | Contagem regressiva antes de liberar lote de entregas no terminal. |

## Enunciado

- Implemente contagem(n) recursiva imprimindo de N ate 0.
- Ao final exiba "Fim da contagem".

## Como executar

```bash
cd "63_recursao_contagem_regressiva"
python main.py
```
