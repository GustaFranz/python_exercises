# 58 - Regex: mascarar CPF

## Objetivo

Mascare CPF 123.456.789-00 para ***.***.***-00 com re.sub.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Clinica BemViver |
| **Setor** | Saude / LGPD |
| **Solicitacao** | Exibir CPF mascarado em telas de recepcao. |

## Enunciado

- Mascare CPF 123.456.789-00 para ***.***.***-00 com re.sub.

## Como executar
