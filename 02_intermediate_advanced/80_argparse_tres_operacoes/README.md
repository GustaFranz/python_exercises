# 80 - Argparse: ferramenta com 3 operacoes

## Objetivo

Integrar argparse com tres subcomandos em ferramenta unica.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | LimpezaDados Servicos |
| **Setor** | Tratamento de dados |
| **Solicitacao** | CLI interna para contar, filtrar e exportar linhas de arquivo texto. |

## Enunciado

- Crie subcomandos contar, filtrar e exportar.
- Implemente cada operacao em funcao separada.
- Teste com arquivo dados.txt de exemplo.

## Como executar

```bash
cd "80_argparse_tres_operacoes"
python main.py contar --arquivo dados.txt
```
