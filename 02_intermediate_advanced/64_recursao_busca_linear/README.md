# 64 - Recursao: busca linear em lista

## Objetivo

Buscar indice de um valor em lista usando recursao.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Edutech Brasil |
| **Setor** | Educacao / plataforma |
| **Solicitacao** | Localizar matricula em lista de alunos exportada do sistema legado. |

## Enunciado

- Implemente buscar(lista, alvo, indice=0) recursiva.
- Teste com valor existente (308) e inexistente (999).

## Como executar

```bash
cd "64_recursao_busca_linear"
python main.py
```
