# 69 - Closure: relatorio por turma

## Objetivo

Gerar funcao de relatorio com config de turma capturada.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Secretaria Digital |
| **Setor** | Educacao / secretaria |
| **Solicitacao** | Gerar cabecalho padrao de relatorio para cada turma. |

## Enunciado

- Implemente criar_gerador_relatorio(turma, professor).
- Gere 2 relatorios com titulos diferentes para turma "7B".

## Como executar

```bash
cd "69_closure_relatorio_turma"
python main.py
```
