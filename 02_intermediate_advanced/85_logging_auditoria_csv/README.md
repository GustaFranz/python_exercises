# 85 - Logging: auditoria de importacao CSV

## Objetivo

Auditar importacao CSV com log estruturado de sucesso e rejeicao.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | GestaoPro RH |
| **Setor** | Recursos humanos |
| **Solicitacao** | Auditar importacao de funcionarios com registro de linhas rejeitadas. |

## Enunciado

- Importe funcionarios.csv registrando sucesso e rejeicao no log.
- Exiba totais importado e rejeitado.

## Como executar

```bash
cd "85_logging_auditoria_csv"
python main.py
```
