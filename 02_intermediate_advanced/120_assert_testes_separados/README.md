# 120 - Assert: arquivo testes separado

## Objetivo

Separar testes em testes.py importando modulo local.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | DevEscola Labs |
| **Setor** | Educacao / qualidade |
| **Solicitacao** | Padronizar testes do modulo de conversao de temperatura da aula de ciencias. |

## Estrutura de arquivos

```
120_assert_testes_separados/
├── main.py
├── calculos.py
└── testes.py
```

## Enunciado

- Implemente celsius_para_fahrenheit em calculos.py.
- Crie testes.py com asserts importando calculos.
- Execute python testes.py para validar.

## Como executar

```bash
cd "120_assert_testes_separados"
python testes.py
python main.py
```
