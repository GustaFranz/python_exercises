# 116 - Introducao a testes com assert

## Objetivo

Validar funcao pura de media com assertions.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | DevEscola Labs |
| **Setor** | Educacao / qualidade |

- Exiba "Todos os testes passaram." ao final.
