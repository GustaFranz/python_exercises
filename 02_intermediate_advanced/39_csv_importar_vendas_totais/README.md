# 39 - CSV: importar vendas e totais

## Objetivo

Importar CSV de vendas e calcular metricas comerciais.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Mercado Bom Preco |
| **Setor** | Varejo |
| **Solicitacao** | Importar vendas do caixa e calcular faturamento do dia. |

## Enunciado

vendas.csv: produto,quantidade,preco_unitario
caneta,10,2.50
caderno,5,12.00
Leia CSV, calcule subtotal por linha e total geral.
Exiba tabela e totais: itens vendidos, faturamento total.

## Como executar

```bash
cd "39_csv_importar_vendas_totais"
python main.py
```
