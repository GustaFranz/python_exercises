# 94 - Counter: tickets de suporte

## Objetivo

Gerar relatorio de ocorrencias por categoria de ticket.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Edutech Brasil |
| **Setor** | Educacao / suporte |
| **Solicitacao** | Resumir tickets de suporte da plataforma por categoria. |

## Enunciado

- Conte tickets por categoria com Counter.
- Exiba relatorio ordenado por quantidade.

## Como executar

```bash
cd "94_counter_tickets_suporte"
python main.py
```
