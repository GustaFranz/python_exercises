# 04 - List comprehension: engajamento de posts

## Objetivo

Montar relatorio aplicado com list comprehension.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Rede Social EduConnect |
| **Setor** | Midia / comunicacao escolar |
| **Solicitacao** | Identificar posts com alto engajamento para destaque no feed. |

## Enunciado

posts = [
    {"titulo": "Feira de Ciencias", "curtidas": 120},
    {"titulo": "Aviso Prova", "curtidas": 45},
    {"titulo": "Projeto Leitura", "curtidas": 200},
    {"titulo": "Recesso", "curtidas": 30},
]
Limite de destaque: curtidas >= 100
Gere lista de titulos em destaque com list comprehension.
Exiba relatorio: total de posts, total em destaque e titulos destacados.

## Como executar

```bash
cd "04_list_comprehension_engajamento_posts"
python main.py
```
