# 101 - Introducao a heranca

## Objetivo

Criar hierarquia Pessoa -> Aluno e Professor com heranca simples.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Edutech Brasil |
| **Setor** | Educacao / cadastro |
| **Solicitacao** | Unificar cadastro de pessoas com tipos especificos aluno e professor. |

## Visao do bloco (exercicios 101 a 105)

Topico **Heranca simples**: reutilizar atributos e metodos da classe pai.

| # | Foco |
|---|------|
| 101 | Introducao + Pessoa, Aluno, Professor |
| 102 | Sobrescrever metodo apresentar() |
| 103 | Usar super() no construtor |
| 104 | Hierarquia de descontos por tipo de cliente |
| 105 | Sistema de RH com 3 classes relacionadas |

## Enunciado

- Crie Pessoa, Aluno(Pessoa) e Professor(Pessoa).
- Instancie um aluno e um professor e exiba com print.

## Como executar

```bash
cd "101_introducao_heranca"
python main.py
```
