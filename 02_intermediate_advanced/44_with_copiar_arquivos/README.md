# 44 - With open: copiar entre arquivos

## Objetivo

Crie consulta_ana.txt com 3 linhas.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Clinica BemViver |
| **Setor** | Saude / prontuario |
| **Solicitacao** | Copiar resumo de consulta para pasta de backup local (arquivo texto). |

## Enunciado

- Crie consulta_ana.txt com 3 linhas.
- Copie para backup_consulta_ana.txt com with open.
- Confirme que o backup e identico ao original.

## Como executar

```bash
cd "44_with_copiar_arquivos"
python main.py
```
