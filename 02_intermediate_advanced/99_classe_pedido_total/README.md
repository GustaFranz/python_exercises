# 99 - Classe Pedido: itens e total

## Objetivo

Montar pedido com lista de itens e calcular total.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | LogiRapida |
| **Setor** | Logistica / pedidos |
| **Solicitacao** | Calcular valor total de pedidos de material para entrega escolar. |

## Enunciado

- Crie classe Pedido com adicionar_item() e total().
- Monte pedido com 2 itens e exiba total.

## Como executar

```bash
cd "99_classe_pedido_total"
python main.py
```
