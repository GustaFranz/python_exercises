# 95 - Counter: dashboard por categoria

## Objetivo

Montar dashboard textual de metricas por categoria.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | MonitoraTI |
| **Setor** | Infraestrutura / NOC |
| **Solicitacao** | Painel textual de alertas por tipo para reuniao de status. |

## Enunciado

- Monte dashboard textual com Counter e percentuais.
- Exiba cada categoria com quantidade e percentual do total.

## Como executar

```bash
cd "95_counter_dashboard_categorias"
python main.py
```
