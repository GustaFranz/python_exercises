# 115 - Refatoracao: script longo

## Objetivo

Refatorar script extenso em funcoes e fluxo organizado.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | AgroEscola |
| **Setor** | Educacao / campo |
| **Solicitacao** | Reorganizar controle de estoque do viveiro escolar antes da safra. |

## Estrutura de arquivos

```
115_refatoracao_script_longo/
├── README.md
├── main.py      # sua versao refatorada
└── legado.py    # script monolitico de referencia
```

## Enunciado

- Estude o script monolitico em legado.py.
- Refatore em main.py com funcoes e menu simples.
- Preserve regras de estoque do script original.

## Como executar

```bash
cd "115_refatoracao_script_longo"
python main.py
```
