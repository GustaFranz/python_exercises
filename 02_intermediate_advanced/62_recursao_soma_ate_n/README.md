# 62 - Recursao: soma de 1 ate N

## Objetivo

Somar 1 + 2 + ... + N usando recursao.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | FinEdu Carteira |
| **Setor** | Financeiro educacional |
| **Solicitacao** | Somar parcelas mensais acumuladas para simulacao de plano. |

## Enunciado

- Implemente soma_ate(n) recursiva.
- Exiba resultados para n = 1, 5 e 10.

## Como executar

```bash
cd "62_recursao_soma_ate_n"
python main.py
```
