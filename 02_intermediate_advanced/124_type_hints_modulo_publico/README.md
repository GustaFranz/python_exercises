# 124 - Type hints: modulo publico

## Objetivo

Criar modulo com type hints em todas as funcoes publicas.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | FinEdu Carteira |
| **Setor** | Financeiro educacional |
| **Solicitacao** | Publicar modulo de calculo de mensalidade com contrato tipado. |

## Estrutura de arquivos

```
124_type_hints_modulo_publico/
├── main.py
└── mensalidades.py
```

## Enunciado

- Implemente mensalidades.py com 3 funcoes tipadas.
- Use o modulo em main.py e exiba resumo.

## Como executar

```bash
cd "124_type_hints_modulo_publico"
python main.py
```
