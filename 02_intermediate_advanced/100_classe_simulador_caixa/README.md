# 100 - Classe: simulador de caixa

## Objetivo

Simular caixa escolar com duas classes cooperando.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Loja Virtual Escolar |
| **Setor** | Varejo / PDV |
| **Solicitacao** | Simular ponto de venda do bazar escolar com registro de vendas. |

## Enunciado

- Crie ItemVenda e Caixa com metodos subtotal e total_dia.
- Simule 3 vendas e exiba resumo do caixa.

## Como executar

```bash
cd "100_classe_simulador_caixa"
python main.py
```
