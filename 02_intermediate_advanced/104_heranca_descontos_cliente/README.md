# 104 - Heranca: descontos por tipo

## Objetivo

Aplicar descontos diferentes por tipo de cliente com heranca.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Loja Virtual Escolar |
| **Setor** | Varejo / precificacao |
| **Solicitacao** | Calcular preco final conforme tipo de cliente no bazar escolar. |

## Enunciado

- Crie Cliente, ClienteEscola e ClienteParceiro com desconto() distinto.
- Calcule preco_final para valor 100 em cada tipo.

## Como executar

```bash
cd "104_heranca_descontos_cliente"
python main.py
```
