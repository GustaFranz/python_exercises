# 68 - Closure: filtro com limite

## Objetivo

Filtrar valores acima de limite capturado em closure.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Edutech Brasil |
| **Setor** | Educacao / avaliacoes |
| **Solicitacao** | Filtrar alunos com nota acima do corte da turma. |

## Enunciado

- Implemente criar_filtro_minimo(limite) retornando funcao aceitar(nota).
- Filtre lista de notas com limite 7.0.

## Como executar

```bash
cd "68_closure_filtro_limite"
python main.py
```
