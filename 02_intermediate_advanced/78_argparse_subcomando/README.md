# 78 - Argparse: subcomando simples

## Objetivo

Criar subcomandos listar e exportar na mesma CLI.

## Demanda

| Campo | Detalhe |
|-------|---------|
| **Empresa** | Secretaria Digital |
| **Setor** | Educacao / secretaria |
| **Solicitacao** | Ferramenta unica para listar ou exportar alunos por turma. |

## Enunciado

- Crie subcomandos listar e exportar com argparse.
- Cada subcomando recebe --turma; exportar tambem recebe --arquivo.

## Como executar

```bash
cd "78_argparse_subcomando"
python main.py listar --turma 7B
```
