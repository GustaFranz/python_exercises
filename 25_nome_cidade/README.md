# 25 - Nome da cidade

## 🎯 Objetivo

Verificar se o nome de uma cidade comeca com SANTO.

## 🧠 O que pratiquei

- Busca com `.find()` e fatiamento de string
- Comparacao ignorando maiusculas e minusculas
- Duas versoes de solucao no arquivo (comentadas)

## 🧾 Código

Arquivo: [`main.py`](https://github.com/GustaFranz/exercicios_python/blob/main/25_nome_cidade/main.py)

## 🚀 Como executar

Descomente uma das versoes no `main.py` antes de executar.

```bash
cd "25_nome_cidade"
python main.py
```
