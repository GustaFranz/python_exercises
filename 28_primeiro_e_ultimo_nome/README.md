# 28 - Primeiro e ultimo nome

## 🎯 Objetivo

Exibir o primeiro e o ultimo nome de uma pessoa.

## 🧠 O que pratiquei

- Divisao de texto com `.split()`
- Acesso por indice em listas
- Uso de `len()` para pegar o ultimo item
