# 13 - Converter Celsius para Fahrenheit

## 🎯 Objetivo

Transformar graus Celsius informados em Fahrenheit.

## 🧠 O que pratiquei

- Formula `(celsius * 1.8) + 32`
- Conversao de temperatura
- Mensagem final com unidade correta
