# 40 - Classificador de palavras

## Objetivo

Classificar palavras por tamanho usando listas e contadores.

## Enunciado

Le 10 palavras e separa em curtas (ate 4 letras), medias (5 a 8) e longas (9 ou mais), exibindo a quantidade em cada categoria.

## 🚀 Como executar

```bash
cd "40_classificador_palavras"
python main.py
```
