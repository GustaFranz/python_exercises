# 04 - Funcao format

## 🎯 Objetivo

Calcular a soma de dois numeros e exibir o resultado com `.format()`.

## 🧠 O que pratiquei

- Metodo `.format()` em strings
- Substituicao de placeholders `{}`
- Soma de inteiros lidos do teclado
