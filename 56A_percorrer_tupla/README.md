# 56A - Percorrer tupla

## Objetivo

Percorrer uma tupla e contar seus elementos.

## Enunciado

Exiba cada dia da semana de uma tupla e, no final, o total de dias.

## Como executar

```bash
cd "56A_percorrer_tupla"
python main.py
```

## Conceitos trabalhados

- Tupla com valores fixos
- Laço `for` com `enumerate`
- Contagem com `len()`
