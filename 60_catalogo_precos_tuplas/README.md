# 60 - Catalogo de precos com tuplas

## Objetivo

Somar valores e encontrar o item de maior preco em tuplas.

## Enunciado

Cada produto e uma tupla (produto, preco). Exiba o total dos precos e o produto mais caro.

## Como executar

```bash
cd "60_catalogo_precos_tuplas"
python main.py
```

## Conceitos trabalhados

tuplas, acumulador, maximo, enumerate
