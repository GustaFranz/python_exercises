# 08 - Conversao do sistema metrico decimal

## 🎯 Objetivo

Converter um valor em metros para outras unidades de comprimento.

## 🧠 O que pratiquei

- Trabalho com `float`
- Multiplicacao e divisao para conversao
- Relacao entre km, hm, dam, m, dm, cm e mm

## 🧾 Código

Arquivo: [`main.py`](https://github.com/GustaFranz/exercicios_python/blob/main/08_conversao_sistema_metrico_decimal/main.py)

## 🚀 Como executar


```bash
cd "08_conversao_sistema_metrico_decimal"
python main.py
```
