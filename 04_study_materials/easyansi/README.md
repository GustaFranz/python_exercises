# Guia EasyAnsi

Material de apoio em PDF sobre a biblioteca EasyAnsi.

## Regenerar o PDF

```bash
python 04_study_materials/gerar_todos.py
python 04_study_materials/_gerar_previews.py
```

Conteudo em `04_study_materials/gerador/conteudo/easyansi.py`.

## Arquivo

- `EasyAnsi_em_Python.pdf` — guia completo com sintaxe, atalhos, logging e exercicio pratico.
