# 70 - Classificador de palavras por tamanho

## Objetivo

Classificar palavras prontas por tamanho usando listas.

## Enunciado

Separa palavras curtas, medias e longas a partir de uma lista fornecida.

## Como executar

```bash
cd "70_classificador_palavras_tamanho"
python main.py
```

## Conceitos trabalhados

len, classificacao por faixas, listas com elif
