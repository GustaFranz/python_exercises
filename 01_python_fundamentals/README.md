# 01 — Python Fundamentals

Trilha de exercicios progressivos que cobre os fundamentos de Python: do primeiro `print` ate funcoes, estruturas de dados, modulos, bibliotecas padrao, APIs e tratamento de excecoes.

**Total:** 127 exercicios numerados (`01` a `127`, com `56A` e `56B`).

[← Voltar ao README principal](../README.md)

---

## Objetivo desta area

| Item | Descricao |
|------|-----------|
| **Para quem** | Iniciantes e quem quer consolidar a base antes de avancar |
| **O que treina** | Logica, sintaxe, estruturas de dados e boas praticas iniciais |
| **Como usar** | Siga a ordem numerica ou escolha um bloco por tema |
| **Cada exercicio** | Pasta propria com `README.md` (objetivo + enunciado) e `main.py` |

---

## Como executar

```bash
git clone https://github.com/GustaFranz/python_exercises.git
cd python_exercises/01_python_fundamentals
cd 04_funcao_format          # exemplo: troque pelo exercicio desejado
python main.py
```

1. Entre na pasta do exercicio
2. Leia o `README.md` da pasta
3. Execute `python main.py`
