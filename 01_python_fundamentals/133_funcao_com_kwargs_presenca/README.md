# 133 - Funcao com **kwargs: registro de presenca

## Objetivo

Registrar presenca de varios alunos usando `**kwargs` como pares nome=status.

## Enunciado

Crie uma funcao que recebe alunos e status de presenca por `**kwargs` e exibe resumo da aula.

## Como executar

```bash
cd "133_funcao_com_kwargs_presenca"
python main.py
```
