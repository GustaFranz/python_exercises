# 137 - Funcao com *args: media de notas

## Objetivo

Usar `*args` para calcular media com quantidade variavel de notas.

## Enunciado

Crie uma funcao que recebe varias notas, calcula a media e informa se o aluno foi aprovado (media >= 6).

## Como executar

```bash
cd "137_funcao_com_args_media"
python main.py
```
