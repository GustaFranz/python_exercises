# 65 - Ordenar lista

## Objetivo

Ordenar uma lista em ordem crescente e decrescente.

## Enunciado

Ordene uma lista de numeros em ordem crescente e depois em ordem decrescente.

## Como executar

```bash
cd "65_ordenar_lista"
python main.py
```

## Conceitos trabalhados

sorted, reverse, ordenacao sem alterar original
