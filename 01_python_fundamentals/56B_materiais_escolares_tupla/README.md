# 56B - Materiais escolares em tupla

## Objetivo

Complemento do exercicio 56A: cadastrar materiais e exibi-los numerados a partir de uma tupla.

## Enunciado

Cadastre materiais escolares informados pelo usuario, armazene-os em tupla, exiba a lista numerada e mostre a quantidade total.

## Como executar

```bash
cd "56B_materiais_escolares_tupla"
python main.py
```

## Conceitos trabalhados

- Entrada de dados com `input()` e loop
- Conversao de lista para tupla com `tuple()`
- `enumerate` para numeracao na saida
