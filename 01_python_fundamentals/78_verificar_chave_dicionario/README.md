# 78 - Verificar chave no dicionario

## Objetivo

Verificar se uma chave existe no dicionario e acessar valor com seguranca.

## Enunciado

Verifique se um aluno digitado pelo usuario existe no dicionario de notas. Use get() para evitar erro.

## Como executar

```bash
cd "78_verificar_chave_dicionario"
python main.py
```
