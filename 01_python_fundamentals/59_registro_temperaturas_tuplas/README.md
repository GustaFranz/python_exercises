# 59 - Registro de temperaturas com tuplas

## Objetivo

Encontrar o maior e o menor valor em uma lista de tuplas.

## Enunciado

Cada registro e uma tupla (dia, temperatura). Exiba o dia mais quente e o dia mais frio.

## Como executar

```bash
cd "59_registro_temperaturas_tuplas"
python main.py
```
## Conceitos trabalhados

maximo e minimo em tuplas
