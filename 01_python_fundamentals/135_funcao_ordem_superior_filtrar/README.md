# 135 - Funcao de ordem superior: filtrar com lambda interna

## Objetivo

Criar uma funcao que define uma lambda internamente para filtrar dados.

## Enunciado

Crie uma funcao que recebe uma lista de notas e um limite, usando lambda interna para retornar apenas as notas aprovadas.

## Como executar

```bash
cd "135_funcao_ordem_superior_filtrar"
python main.py
```
