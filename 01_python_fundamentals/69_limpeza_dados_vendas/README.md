# 69 - Limpeza de dados de vendas

## Objetivo

Limpar dados invalidos de vendas antes da analise.

## Enunciado

Remove valores invalidos da lista de vendas e calcula total, maior, menor e media das vendas validas.

## Como executar

```bash
cd "69_limpeza_dados_vendas"
python main.py
```
