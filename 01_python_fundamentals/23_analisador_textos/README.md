# 23 - Analisador de textos

## Objetivo

Manipular strings (maiusculas, minusculas e contagem de letras).

## Enunciado

Le nome completo e mostra versao maiuscula/minuscula, total de letras sem espacos e letras do primeiro nome.

## 🚀 Como executar

```bash
cd "23_analisador_textos"
python main.py
```
