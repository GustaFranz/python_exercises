# 58 - Boletim de medias com tuplas

## Objetivo

Calcular a media de cada aluno a partir de uma lista de tuplas.

## Enunciado

Cada aluno e uma tupla (nome, nota1, nota2). Calcule e exiba a media de cada aluno.

## Como executar

```bash
cd "58_boletim_medias_tuplas"
python main.py
```
## Conceitos trabalhados

media, tuplas, classificacao visual
