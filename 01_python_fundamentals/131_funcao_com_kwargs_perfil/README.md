# 131 - Funcao com **kwargs: perfil do aluno

## Objetivo

Criar uma funcao que recebe informacoes nomeadas com `**kwargs`.

## Enunciado

Crie uma funcao que monta e exibe o perfil de um aluno a partir de pares chave-valor enviados na chamada.

## Como executar

```bash
cd "131_funcao_com_kwargs_perfil"
python main.py
```
