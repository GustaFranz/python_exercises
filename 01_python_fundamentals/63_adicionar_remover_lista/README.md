# 63 - Adicionar e remover itens da lista

## Objetivo

Adicionar e remover itens de uma lista.

## Enunciado

Adicione um novo item no fim da lista de compras e remova um item existente. Exiba a lista final.

## Como executar

```bash
cd "63_adicionar_remover_lista"
python main.py
```

## Conceitos trabalhados

append, remove, mutabilidade de listas
