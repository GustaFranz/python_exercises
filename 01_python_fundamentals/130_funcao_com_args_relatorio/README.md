# 130 - Funcao com *args: relatorio do aluno

## Objetivo

Combinar parametro fixo com `*args` para montar um relatorio escolar.

## Enunciado

Crie uma funcao que recebe o nome do aluno e varias notas com `*args`, exibindo relatorio com media e maior nota.

## Como executar

```bash
cd "130_funcao_com_args_relatorio"
python main.py
```
