# 51 - Catalogo de produtos fixos

## Objetivo

Consultar um catalogo imutavel de produtos com tuplas.

## Enunciado

Armazena produtos como tuplas (codigo, nome, preco), exibe o catalogo completo e permite apenas consulta.

## Como executar

```bash
cd "51_catalogo_produtos_fixos"
python main.py
```
## Conceitos trabalhados

lista de tuplas, consulta, busca

## Status

Resolvido.

Dependencias: easyansi.
