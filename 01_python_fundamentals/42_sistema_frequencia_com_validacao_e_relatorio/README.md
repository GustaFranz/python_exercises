# 42 - Sistema de frequencia com relatorio

## Objetivo

Controlar frequencia escolar com relatorio completo e repeticao do sistema.

## Enunciado

Registra presenca (P) ou falta (F) em 12 aulas, calcula percentual, identifica maiores sequencias consecutivas, exibe relatorio e permite analisar outro aluno.

## 🚀 Como executar

```bash
cd "42_sistema_frequencia_com_validacao_e_relatorio"
python main.py
```
