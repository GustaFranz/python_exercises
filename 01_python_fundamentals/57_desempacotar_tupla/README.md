# 57 - Desempacotar tupla

## Objetivo

Desempacotar os valores de uma tupla em variaveis.

## Enunciado

Separe os dados de uma pessoa (nome, idade e cidade) em variaveis e exiba cada uma.

## Como executar

```bash
cd "57_desempacotar_tupla"
python main.py
```

desempacotamento, lista de tuplas, enumerate
