# 24 - Separador de digitos

## 🎯 Objetivo

Separar unidade, dezena, centena e milhar de um numero inteiro.

## 🧠 O que pratiquei

- Divisao inteira (`//`) e modulo (`%`)
- Decomposicao numerica
- Formatacao da saida com `.format()`

Arquivo: [`main.py`](https://github.com/GustaFranz/exercicios_python/blob/main/24_separador_digitos/main.py)

## 🚀 Como executar


```bash
cd "24_separador_digitos"
python main.py
```
