# 14 - Permissao de passagem de veiculos

## Objetivo

Aplicar condicionais para autorizar ou negar passagem de veiculo.

## Enunciado

Pergunta o peso do veiculo; entre 0,9 e 1,8 toneladas permite passagem; fora disso, pede retorno.

## 🚀 Como executar

```bash
cd "14_permissao_passagem_veiculos"
python main.py
```
