# 02 - Soma de dois numeros

## 🎯 Objetivo

Ler dois numeros inteiros e mostrar a soma de forma clara.

## 🧠 O que pratiquei

- Leitura de numeros com `input()` e `int()`
- Operacao aritmetica basica
- Formatacao com f-string

## 🧾 Código

Arquivo: [`main.py`](https://github.com/GustaFranz/exercicios_python/blob/main/02_%20soma_de_dois_numeros/main.py)

## 🚀 Como executar


```bash
cd "02_ soma_de_dois_numeros"
python main.py
```
