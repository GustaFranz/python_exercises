# 32 - Calculo do preco da passagem

## 🎯 Objetivo

Calcular o valor da passagem conforme a distancia da viagem.

## 🧠 O que pratiquei

- Preco diferente para viagens curtas e longas
- Condicional com limite de 200 km
- Calculo com valores decimais

## 🧾 Código

Arquivo: [`main.py`](https://github.com/GustaFranz/exercicios_python/blob/main/32_calculo_preco_passagem/main.py)

## 🚀 Como executar


```bash
cd "32_calculo_preco_passagem"
python main.py
```
