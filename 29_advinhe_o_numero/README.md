# 29 - Advinhe o numero

## 🎯 Objetivo

Desafiar o usuario a adivinhar um numero sorteado pelo computador.

## 🧠 O que pratiquei

- Geracao aleatoria com `randint()`
- Comparacao entre valor digitado e sorteado
- Mensagens de acerto e erro

## 🧾 Código

Arquivo: [`main.py`](https://github.com/GustaFranz/exercicios_python/blob/main/29_advinhe_o_numero/main.py)

## 🚀 Como executar


```bash
cd "29_advinhe_o_numero"
python main.py
```
