# 23 - Analisador de textos

## 🎯 Objetivo

Analisar um nome completo com maiusculas, minusculas e contagem de letras.

## 🧠 O que pratiquei

- Metodos `.upper()`, `.lower()` e `.split()`
- Contagem de caracteres com `len()`
- Duas abordagens de solucao no arquivo (comentadas)

## 🧾 Código

Arquivo: [`main.py`](https://github.com/GustaFranz/exercicios_python/blob/main/23_analisador_textos/main.py)

## 🚀 Como executar

Descomente uma das versoes no `main.py` antes de executar.

```bash
cd "23_analisador_textos"
python main.py
```
