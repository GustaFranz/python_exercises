# 66 - Verificar item na lista

## Objetivo

Verificar se um elemento existe em uma lista usando o operador in.

## Enunciado

Verifique se uma fruta digitada pelo usuario esta ou nao na lista de frutas disponivel.

## Como executar

```bash
cd "66_verificar_item_lista"
python main.py
```

operador in, append, loop com break
